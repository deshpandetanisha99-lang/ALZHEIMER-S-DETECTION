ALZHEIMER'S DETECTION
Early Detection of Alzheimer's from Patient Data

OVERVIEW
The work uses machine learning to classify longitudinal patient data with a Random Forest Classifier that predicts whether a patient is likely to develop Dementia or remain Non-Demented.
Clinical features include cognitive scores and demographic information that are part of the dataset.
The aim is to develop a reproducible data-driven pipeline for early detection and support.

FEATURES
1)Load and clean longitudinal clinical records(.csv)
2.Handle missing values automatically
3)Encode categorical variables
4)Split dataset into training and testing groups
5)Train a Random Forest Classifier
6)Evaluate accuracy, confusion matrix and classification report
7)Save model for future predictions

2)Google ColaB
Libraries:
4)pandas
